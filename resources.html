Resources

Code and datasets from our published work, freely available.

github.com/okazawagouki/Okazawa_Cell_2021

LIP neural data during face categorization task

Data and code used in Okazawa et al. (2021) Cell. Neural population recordings from monkey parietal cortex during a face categorization task, with analysis code for representational geometry.

github.com/okazawagouki/Okazawa_NatComm_2018

Psychophysical kernels of drift-diffusion model

Code to generate simulated psychophysical kernels for drift-diffusion model variants, from Okazawa et al. (2018) Nature Communications.

github.com/okazawagouki/minPS

Minimum visual texture statistics (minPS)

Code to calculate the minimum texture statistics to fit V4 neural data, from Okazawa et al. (2015) PNAS.

shitsukan-db.jp/public/texture-db/

Large-scale synthetic texture database

~10,000 synthetic texture images created for Okazawa et al. (2015) PNAS. Freely available for download.
